# 风格库

仅在用户请求相应风格或提供新的剪辑录屏供学习时读取本文件。新增条目必须是跨内容可复用的操作模式，不得包含原始媒体、人物信息或原文文案。

## 新条目格式

## Variant: 风格名称

- Avoid when:

## Variant: information-dense talking head

- Use when: 口播包含连续观点、术语、数字或结论，且需要快速但清晰地传达信息。
- Sequence: 先按观点分段完成粗剪；校正逐句字幕；在核心结论前后加入简短标题或重点文字。
- Typography and placement: 使用清晰的高对比字幕；顶部标题短且不遮挡人脸；重点文字只强调关键词、数字或结论。
- Optional tracks: 仅在能解释口播内容时加入截图、图表或其他辅助视觉轨道。
- Avoid when: 素材本身节奏较慢、观点需要铺垫，或增加文字会造成字幕拥挤时。
- Evidence: 初始剪映录屏展示了逐句字幕修正、短顶部标题和选择性辅助轨道。
